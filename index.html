REKOMENDASI

Template Resmi CBT v2.0

Mendukung soal Benar/Salah & opsi gambar

Pilih / Unggah File Excel Hasil Pengisian

Pastikan struktur kolom tidak diubah (xlsx/xls)

Petunjuk Kolom Excel:

  1. Kolom 1: Tipe Soal (Isikan: Pilihan Ganda atau Benar Salah).
  2. Kolom 2: Pertanyaan ujian (Gunakan $...$ untuk rumus fungsi).
  3. Kolom 3-6: Pilihan Jawaban A, B, C, D (Kosongkan C & D jika tipe Benar Salah).
  4. Kolom 7-10: URL Gambar untuk masing-masing opsi A, B, C, D (opsional).
  5. Kolom 11: Kunci Jawaban (A, B, C, atau D).
  6. Kolom 12: Penjelasan pembahasan (opsional).
  7. Kolom 13: URL Gambar Utama pendukung soal (opsional).
